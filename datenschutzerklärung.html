Datenschutz

1. Datenschutz auf einen Blick

Diese Datenschutzerklärung informiert dich darüber, wie deine personenbezogenen Daten verarbeitet werden, wenn du meine Website besuchst.

2. Hosting und CDNs

Ich nutze Tailwind CSS über ein CDN (Content Delivery Network). Um das Design der Website korrekt anzuzeigen, wird deine IP-Adresse technisch bedingt an den Server des Anbieters (tailwindcss.com) übertragen.

3. Kontaktformular & Anfragen

Wenn du mir eine Anfrage über das Formular sendest, werden deine Angaben (Name, E-Mail, Nachricht) zwecks Bearbeitung der Anfrage verarbeitet. Ich nutze hierfür den Dienst FormSubmit (formsubmit.co), der die Daten verschlüsselt an mein Postfach weiterleitet.

4. Google Web Fonts

Diese Seite nutzt Web Fonts von Google zur einheitlichen Darstellung von Schriftarten. Beim Laden der Seite verbindet sich dein Browser mit Google-Servern, wobei Google erfährt, dass meine Website über deine IP-Adresse aufgerufen wurde.

5. Deine Rechte

Du hast jederzeit das Recht, unentgeltlich Auskunft über Herkunft, Empfänger und Zweck deiner gespeicherten personenbezogenen Daten zu erhalten. Du hast außerdem ein Recht auf Berichtigung oder Löschung dieser Daten.
